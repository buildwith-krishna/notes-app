📝 Notes App (CLI) - v3

A simple and efficient command-line Notes App built using Python.
This project allows users to manage notes directly from the terminal with basic CRUD operations.

---

🚀 Features

- Add a note
- View all notes
- Delete a note
- Update a note
- Persistent storage using "notes.txt"
- Input validation and error handling

---

🧠 What’s New in v3

- Added Update Note feature
- Improved input validation
- Better error handling (try-except)
- Cleaner logic and flow

---

📂 Project Structure

notes_app/
├── notes.py      # Main application
├── notes.txt     # Stores all notes
└── README.md     # Project documentation

---

🛠️ How It Works

- Notes are stored line-by-line in "notes.txt"
- Each operation:
  - Reads file
  - Processes in list
  - Writes back

---

▶️ Run

python notes.py

---

⚠️ Limitations

- No timestamps
- No search
- Text file storage
